# https://3.14n.dev

![Deploy](https://github.com/luizfelipefb/3.14n.dev/workflows/Deploy/badge.svg)

## Build

```bash
# serve at localhost:3000
npm run dev

# build for production and launch server
npm run build
npm run start

# generate static project
npm run generate
```

## Deploy

```bash
# deploy to firebase
firebase deploy --project <project_id>
```
